# vim基本用法

## 基本命令
### 编辑命令
```
文本对象            更改        删除        复制
一个单词            cw           dw        yw
两个单词, 不包括标点 2cw或c2W    2dW或d2W   2yW或y2W
后退三个单词        3cb         3db或d3b   3yb或y3b
一整行              cc           dd        yy或Y
到一行结尾          c$或C        d$或D       y$
到一行开头          c0            d0         y0
单个字符            r           x或X         y
五个字符            5s            5x        5y
```
### 光标移动命令
```
左下上右    h j k l
到下一行的第一个字符 +
到上一行的第一个字符 -
到单词结尾 e或E
前进一个单词 w或W
后退一个单词 b或B
到一行结尾 $
到一行开头 0
```

### 其他操作
```
将缓冲区内容放置到光标出 P(放到光标之前) p(放到光标之后)
保存编辑结果, 并离开文件 ZZ
消除所有修改内容,从头开始 e!
```

### 文本创建和操作命令
```
在光标所在位置之前插入文本 i
在光标所在行开头插入文本 I
在光标所在位置后附加文本 a
在光标所在行开头附加文本 A
在光标下一行打开新行 o
在光标上一行打开新行 O
删除一行并替换文本 S
用新文本覆盖现有文本 R
合并当前行与下一行 J
切换当前字符大小写 ~
重复上一个动作 .
撤销上一个动作 u
将一行回复到原来状态 U
```

### 浏览命令
```
往前滚动一屏 Ctrl+F
往后滚动一屏 Ctrl+B
往前移动半屏 Ctrl+D
往后移动半屏 Ctrl+U
往前滚动一行 Ctrl+E
往后滚动一行 Ctrl+Y

```
