# 磁盘容量相关命令

```
查看磁盘剩余空间
df -k        以KB为单位查看磁盘剩余空间
df -m        以MB为单位查看磁盘剩余空间
df -h        用人类(human)可以看懂的方式显示磁盘剩余空间

查看某目录占用空间大小
du -h /home    查看/home占用的空间, 会显示home下所有子文件夹和文件占用的空间
du -sh /home   只显示home根目录占用的空间(也就是home内所有文件占用的总空间)
# -h是以人类可读的方式显示 -s是:显示目录占用的磁盘空间大小，不要显示其下子目录和文件占用的磁盘空间大小

```
