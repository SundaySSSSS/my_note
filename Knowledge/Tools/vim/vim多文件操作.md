# vim多文件操作
## buffer
当文件打开时, 会在内存中创建一个缓冲区(buffer)


### 在vim中打开文件
```
:e 文件名
```

### buffer切换
```
:ls    # 列举当前缓冲区, 然后使用:b n跳转到第n个缓冲区

```

## 多窗口操作
### 打开多个窗口
```
# 水平分割
<Ctrl + w> s 
:sp

# 垂直分割
<Ctrl + w> v
:vs
```

### 切换窗口
```
<Ctrl + w>w # 在窗口中循环切换
<Ctrl + w>h # 切换到左边的窗口
<Ctrl + w>j # 切换到下边的窗口
<Ctrl + w>k # 切换到上边的窗口
<Ctrl + w>l # 切换到右边的窗口
```

### 移动窗口
```
<Ctrl + w>R # 将窗口移动到右边
```
